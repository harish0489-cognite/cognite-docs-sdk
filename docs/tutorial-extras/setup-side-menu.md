# Setup - Side Menu

***Side Menu*** has a list of items, lables and its assoicated links.

### Code Block for Side Menu


```tsx title="menu.json"
{
    "label": "Home",
    "link": "/",
    "icon": "fas fa-home"
},
{
    "label": "Assets",
    "link": "/assets",
    "icon": "fas fa-dollar-sign"
},

```
