# Time Series

Fetch results based on Start Time and End Time.

## Code Blocks


```tsx title="src/components/assets.tsx"
 async function fetchResult() {
   const models3D = await client.events.list({ 
      filter: { 
        startTime: { min: new Date('1 jan 2021') }, 
        endTime: { max: new Date('1 dec 2021') } 
        } }
    );
```
